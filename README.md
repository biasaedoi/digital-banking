Digital Banking System

Digital Banking System adalah aplikasi simulasi perbankan modern yang dibangun menggunakan Laravel (backend) dan React.js (frontend).
Aplikasi ini memungkinkan pengguna untuk:
- Melihat saldo akun
- Melakukan transfer antar rekening
- Melihat riwayat transaksi
- Autentikasi pengguna (login/register)
- Mengelola data akun dan transaksi secara real-time
